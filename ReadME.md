### Build Guide
SETUP GUIDE FOR VISUAL STUDIO

- Requires at least Windows Kits 10.0.22621.0. Try installing the most up-to-date version of VS2022.

- Clone repo

- Right-click aZeroEditor folder and choose "Open with Visual Studio"

- If the top-level directory doesn't have an "out" folder, CTRL+S the top-level CMakeLists.txt in Visual Studio

- Build all with F7

- Run with F5
